FORMULARIOS

  • Los formularios se ponen dentro de la etiqueta form, que puede servir para enviar los datos establecidos a un servidor:
    • Para enviar los datos a un servidor es necesatio tener un "action="NombreDelServidor"" y un "method="getOpost"", además de que todos los imputs tienen que tenér un name establecido que será el nombre (name="nombre" en el input) con el que interprete la información el servidor, en el method se pueden poner dos maneras:
      • get: Envia la información en una URL por lo que lo pueden ver todos.
      • post: Envia la información en el Response Body (Cuerpo), no se puede ver la respuesta
  • Se utiliza "fieldset" para agruparlo todo en una caja y legend para darle un titulo
    EjemploComo aquí
  • Se utiliza "input" para crear la mayoría de elementos de un formulario, los hay de muchos tipos
  • Se utiliza "label" para poner un texto antes o despues (pero en la misma linea) vinculandolo con un "id"

Los input de tipo radio tienen que tenér el mismo name para que solo deje establecer 1 al mismo riempo, y el label va abajo, también en el checkbox (si en un checkbox se pone "checked" automaticamente saldrá marcado)








Para poner una lista de opciones desplegable se usa la etiqueta "select", las opciónes se ponen con "option" y los grupos de opcones con optgroup (estos se acompañan de un "label" para establecér el nomre del grupo).
El "value" establece que se le enviará a el servidor como información con el tipo del "name"

Los botones se ponen con la etiqueta "button" y su tipo se establece con type, pueden ser submit o reset
