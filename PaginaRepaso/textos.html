En el campo de head se ponen configuraciónes css y en el body html.
Estas etiquetas se pueden convinar siempre que su inicio y cierre estén bien establecidos.

  • Para poner textos de distintos formatos se usa "h1,h2,h3..." por ej el titulo es un h1
  • Para poner un comentario para los programadores se pone, solo se ve en el VSC
  • Para generar muchas palabras para probar cosas varias se utiliza "lorem" y justo despues la cantidad de palabras que quietes generar.
    Ej: lorem10= Lorem ipsum dolor sit amet consectetur adipisicing elit. Minus, temporibus.
  • Si quieres poner un simbolo especifico se utiliza: & nombre y ";"
    Ej: &+copy; = ©
  • La etiqueta "br" salta de linea y no se tiene porque cerrar
  • La etiqueta "p" sirve para pnoner un parrafo, viene bien ya que deja espacio como si pusieras "br" y facilita la edición del texto en css
  • La etiqueta "hr" sirve para poner una linea:
  • La etiqueta "span" sirve para delimitar espacios para modifcarlos con cosas como un style, id, class.
    Ej: Rajoy al poder
  • La etiqueta "u" pone en subrallado el texto recogido: ejemplo ("ins" hace lo mismo)
  • La etiqueta "i" pone en cursiva el texto recogido: ejemplo (También lo hace "em" y
    "address"
    )
  • La etiqueta "b" pone en negrita el texto recogido: ejemplo
  • La etiqueta "small" hace pequeño el texto recogido: ejemplo
  • La etiqueta "big" hace grande el texto recogido: ejemplo
  • La etiqueta "del" tacha el texto recogido: ejemplo
  • La etiqueta "sub" hace pequeño y baja el texto recogido: H2O
  • La etiqueta "sup" hace pequeño y sube el texto recogido: 32=9
  • La etiqueta "blockquote" sirve para citar a alguien en un texto, lo de "cite" es para poner la persona, pero no es obligatorio
